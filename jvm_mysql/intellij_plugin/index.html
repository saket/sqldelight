    IntelliJ Plugin

    The IntelliJ plugin provides language-level features for .sq files, including: - Syntax highlighting - Refactoring/Find usages - Code autocompletion - Generate Queries files after edits - Right click to copy as valid SQLite - Compiler errors in IDE click through to file

    It can be installed from Android Studio by navigating
    Android Studio -> Preferences -> Plugins -> Browse repositories -> Search for SQLDelight
